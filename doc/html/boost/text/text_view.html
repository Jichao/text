Description

A reference to a constant contiguous sequence of char. The sequence is assumed to be UTF-8 encoded, though it is possible to construct a sequence which is not.

text_view public construct/copy/destruct

  1. text_view() noexcept;

    Default ctor.

    This function is constexpr.

    Postconditions:

    data() == nullptr && size() == 0 

  2. text_view(char const * c_str);

    Constructs a text_view from a null-terminated C string. The UTF-8 encoding is checked only at the beginning and end of the string, to prevent slicing of code points. To fully check the encoding, use checked_encoding().

    This function is constexpr in C++14 and later.

    Requires:

    strlen(c_str) <= max_size()

    Postconditions:

    data() == c_str && size() == strlen(c_str)

    Throws:

    		 std::invalid_argument if the ends of the string are not valid UTF-8. 
  3. text_view(char const * c_str, int len);

    Constructs a text_view from an array of char. The UTF-8 encoding is checked only at the beginning and end of the string, to prevent slicing of code points. To fully check the encoding, use checked_encoding().

    This function is constexpr in C++14 and later.

    Requires:

    len >= 0

    Postconditions:

    data() == c_str && size() == len

    Throws:

    		 std::invalid_argument if the ends of the string are not valid UTF-8. 
  4. text_view(char const * c_str, utf8::unchecked_t) noexcept;

    This function is constexpr in C++14 and later.

    Requires:

    Postconditions:

    data() == c_str && size() == strlen(c_str)

  8. constexpr text_view operator()(int lo, int hi) const;

    Returns a substring of *this, taken from the range of chars at offsets [lo, hi). If either of lo or hi is a negative value x, x is taken to be an offset from the end, and so x + size() is used instead.

    This function is constexpr in C++14 and later.

    These preconditions apply to the values used after size() is added to any negative arguments.

    Requires:

    0 <= lo && lo <= size()

    Requires:

    0 <= hi && lhi <= size()

    Requires:

    lo <= hi

    Throws:

    		 std::invalid_argument if the ends of the string are not valid UTF-8. 
  9. constexpr text_view operator()(int cut) const;

    Returns a substring of *this, taken from the first cut chars when cut => 0, or the last -cut chars when cut < 0.

    This function is constexpr in C++14 and later.

    Requires:

    0 <= cut && cut <= size() || 0 <= -cut && -cut <= size()

    Throws:

    		 std::invalid_argument if the ends of the string are not valid UTF-8. 
  10. constexpr int max_size() const noexcept;

  11. constexpr int compare(text_view rhs) const noexcept;

    Lexicographical compare. Returns a value < 0 when *this is lexicographically less than rhs, 0 if *this == rhs, and a value > 0 if *this is lexicographically greater than rhs.

    This function is constexpr in C++14 and later.
