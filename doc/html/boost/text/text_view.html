  2. text_view(char const * c_str);

    Constructs a text_view from a null-terminated C string. The UTF-8 encoding is checked only at the beginning and end of the string, to prevent slicing of code points. To fully check the encoding, use checked_encoding().

    This function is constexpr in C++14 and later.

    Requires:

    strlen(c_str) <= max_size()

    Postconditions:

    data() == c_str && size() == strlen(c_str)

    Throws:

    		 std::invalid_argument if the ends of the string are not valid UTF-8. 
  3. text_view(char const * c_str, int len);

    Constructs a text_view from an array of char. The UTF-8 encoding is checked only at the beginning and end of the string, to prevent slicing of code points. To fully check the encoding, use checked_encoding().

    This function is constexpr in C++14 and later.

    Requires:

    len >= 0

    Postconditions:

    data() == c_str && size() == len

    Throws:

    		 std::invalid_argument if the ends of the string are not valid UTF-8. 
  4. text_view(text const & t) noexcept;

    Constructs a text_view from a text.

    Postconditions:

    data() == t.begin() && size() == t.size()
