Function template rfind_view

boost::text::rfind_view

Synopsis

// In header: <boost/text/algorithm.hpp>


template<typename CharRange, typename PatternCharRange> 
  constexpr auto rfind_view(CharRange const & r, PatternCharRange const & p);

Description

Returns the last occurance of pattern p within range r as a text_view, or text_view() if p is not found in r. An empty p is always considered to match the end of r.

This function only participates in overload resolution if CharRange and PatternCharRange each model the Char_range concept.

This function is constexpr in C++14 and later.
