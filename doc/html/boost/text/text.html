Description

A mutable contiguous null-terminated sequence of char. The sequence is assumed to be UTF-8 encoded, though it is possible to construct a sequence which is not. Strongly exception safe.

  17. text_view operator()(int lo, int hi) const;

    Returns a substring of *this, taken from the range of chars at offsets [lo, hi). If either of lo or hi is a negative value x, x is taken to be an offset from the end, and so x + size() is used instead.

    These preconditions apply to the values used after size() is added to any negative arguments.

    Requires:

    0 <= lo && lo <= size()

    Requires:

    0 <= hi && lhi <= size()

    Requires:

    lo <= hi

    Throws:

    		 std::invalid_argument if the ends of the string are not valid UTF-8. 
  18. text_view operator()(int cut) const;
