Description

A string_view, repeated count() times. This is useful for representing a single char (e.g. for whitespace padding) or sequence of chars, repeated many times, without allocating storage.

  12. unencoded_rope_view operator()(size_type lo, size_type hi) const;

    Returns a substring of *this, taken from the range of chars at offsets [lo, hi). If either of lo or hi is a negative value x, x is taken to be an offset from the end, and so x + size() is used instead.

    This function is constexpr in C++14 and later.

    These preconditions apply to the values used after size() is added to any negative arguments.

    Requires:

    0 <= lo && lo <= size()

    Requires:

    0 <= hi && lhi <= size()

    Requires:

    lo <= hi

    Requires:
