Description

A reference to a substring of a rope, text, or repeated_text_view. The substring is assumed to be UTF-8 encoded, though it is possible to construct a rope_view which is not.

rope_view public construct/copy/destruct

  1. rope_view() noexcept;

    Default ctor.

    Postconditions:

    size() == 0 && begin() == end() 

  2. rope_view(rope const & r) noexcept;

    Constructs a rope_view covering the entire given rope. The UTF-8 encoding is not checked. 

  3. rope_view(rope const & r, int lo, int hi);

    Constructs a substring of r, taken from the range of chars at offsets [lo, hi). If either of lo or hi is a negative value x, x is taken to be an offset from the end, and so x + size() is used instead. The UTF-8 encoding is checked only at the beginning and end of the sequence, to prevent slicing of code points. To fully check the encoding, use checked_encoding().

    These preconditions apply to the values used after size() is added to any negative arguments.

    Requires:

    0 <= lo && lo <= r.size()

    Requires:

    0 <= hi && lhi <= r.size()

    Requires:

    lo <= hi

    Postconditions:

    size() == r.size() && begin() == r.begin() + lo && end() == r.begin() + hi

    Throws:

    		 std::invalid_argument if the ends of the string are not valid UTF-8.

  9. rope_view operator()(int cut) const;

    Returns a substring of *this, taken from the first cut chars when cut => 0, or the last -cut chars when cut < 0.

    Requires:

    Throws:

    		 std::invalid_argument if the ends of the string are not valid UTF-8. 
  10. size_type max_size() const noexcept;

    Returns the maximum size a text_view can have. 

  11. template<typename Fn> void foreach_segment(Fn && f) const;

    Visits each segment s of the underlying rope and calls f(s). Each segment is a value whose type models a Char_iterator iterator-range. Depending of the operation performed on each segment, this may be more efficient than iterating over [begin(), end()).

    Requires:

    Fn is an Invocable accepting a single argument whose begin and end model Char_iterator. 

  12. int compare(rope_view rhs) const noexcept;

    Lexicographical compare. Returns a value < 0 when *this is lexicographically less than rhs, 0 if *this == rhs, and a value > 0 if *this is lexicographically greater than rhs. 

  13. void swap(rope_view & rhs) noexcept;

    Swaps *this with rhs.
