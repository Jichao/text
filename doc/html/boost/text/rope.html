  13. template<typename Fn> void foreach_segment(Fn && f) const;

    Visits each segment s of *this and calls f(s). Each segment is a string_view or repeated_string_view. Depending of the operation performed on each segment, this may be more efficient than iterating over [begin(), end()).

    Requires:

    Fn is an Invocable accepting a single argument of any of the types listed above. 

  14. bool equal_root(rope rhs) const noexcept;

    Returns true if *this and rhs contain the same root node pointer. This is useful when you want to check for equality between two unencoded_ropes that are likely to have originated from the same initial unencoded_rope, and may have since been mutated.
