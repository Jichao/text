Description

A UTF-8 to UTF-16 converting iterator. Set the ErrorHandler template parameter to control error handling. The default ErrorHandler is use_replacement_character, which simply produces a replacement character.

Iter must be a bidirectional iterator with a 1-byte char value_type.
