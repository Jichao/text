boost::text::utf8::null_sentinel

Synopsis

// In header: <boost/text/utf8.hpp>


struct null_sentinel {
};

Description

A sentinel type that compares equal to a to_utf32_iterator or to_utf16_iterator when the iterator points to a 0.
