Description

A UTF-16 to UTF-8 converting iterator. Set the ErrorHandler template parameter to control error handling. The default ErrorHandler is use_replacement_character, which simply produces a replacement character.

Iter must be a bidirectional iterator with a 2-byte integral value_type.
